# CyberRAG — AI-powered Cybersecurity Knowledge Assistant

**CyberRAG** is a Retrieval-Augmented Generation (RAG) system built with Python, LangChain, FAISS, and FastAPI.  
It allows users to query a curated cybersecurity knowledge base (SQLi, XSS, Phishing, Malware, etc.) and receive **structured, detailed answers generated by an LLM**.

---

## Features

- 🔹 RAG system for fast and accurate answers  
- 🔹 Curated cybersecurity knowledge base (PDFs/TXT files)  
- 🔹 FastAPI backend with `/ask` endpoint for queries  
- 🔹 Easy local deployment with virtual environment  
- 🔹 Extensible — add new cybersecurity topics easily
